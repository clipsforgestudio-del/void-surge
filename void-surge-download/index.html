🔧 Build .EXE in 2 minutes

You need Node.js installed. If you don't have it:

👉 Download Node.js here (free)

Then follow these steps:

1. Unzip the downloaded file
2. Open a terminal in the void-surge folder
3. Run: npm install --save-dev electron electron-builder
4. Run: npm run build:win
5. Get your .EXE in the dist/ folder!

Or just double-click BUILD.bat for automatic setup!
